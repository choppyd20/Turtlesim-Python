# Turtlesim-Python
This is project is using and operating the turtlesim used in the ROS package. You will need to run this software in a Linux environment, Ubuntu for example. The following project is to change the turtle's path from going in a circle to a square using twist and angular funcions. This project was created and ran in ROS Noetic.
